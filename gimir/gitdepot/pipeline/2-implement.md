# Phase 2 — Implement

Read `gitdepot/pipeline/00-context.md`, `gitdepot/DESIGN.md`, and
`gitdepot/WORKMAP.md`.

Make the code match the design where the map says it is missing or diverges.
Build on the code the map found to conform; do not rewrite conforming code. Build
until it compiles.

Write what you changed and why to `gitdepot/IMPL-NOTES.md`. If something in the
design is genuinely unimplementable as written, describe the obstacle there
rather than working around it silently. Return a short summary and the build
status.
